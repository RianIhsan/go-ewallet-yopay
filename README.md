# YoPAY

A Mini E-wallet integrated with a payment gateway.

## Features
- Get Information User
- Authentication (Login, Register)
- Charge Balance
- Get Balance

## Tech Stack
- Golang
- Fiber
- Postgres
- Gorm
- Argon
- Midtrans
